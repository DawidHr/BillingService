A Spring Boot-based web application designed to help users manage their bills efficiently.
The system allows users to add, track, compare, and analyze their bills through an intuitive and user-friendly interface.

Features:

Add and manage bills – users can create, edit, and delete their bills with details such as date, amount, and category.

Generate reports – automatically generate summary reports based on user data (e.g., total expenses per month, top categories).

Compare bills – analyze and compare bills from different time periods to identify trends and savings opportunities.

Data persistence – all information is securely stored in a relational database using JPA/Hibernate.

Authentication – integration with JWT for secure user access.

Technologies Used:
- Java 17
- Spring
- Hibernate
- JWT
- MySQL
- Maven
- Lombok
- Junit
